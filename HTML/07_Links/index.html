Elemento link

Links

O elemento link permite definir a relação entre o documento HTML atual e um recurso externo. Por exemplo, utilizamos o elemento link quando queremos associar um arquivo de estilos a uma página web. Esse elemento exige apenas marcação única, sendo assim, para criar uma relação externa no seu documento aHTML utilizamos apenas a tag .
